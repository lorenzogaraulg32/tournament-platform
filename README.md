### Initial SetUp

First of all, for each microservice, copy the application.yml.example and remove .example
After that you need to configure some fields, like username and password for db.


### Auth-service SetUp
For auth-service you will need to create your set of keys, and save them as public.pem and private.pem, in the dir
/resources/keys/private.pem and /resources/keys/public.pem
